Here is a collection, and something of a set of visual tests, of graphics we use for dashboards, reports, and other visualization projects at Mozilla. The point of this repository is to ensure that everything under js/metrics-graphics.js works properly, so we may use that file for other graph-based projects. The main file, metrics-graphics.js, requires d3, jquery, and bootstrap at the moment. Below are a few examples. The data here is synthetic.

Linked Charts

It can be useful to have two charts linked together—when a rollover in one is triggered, the same rollover is triggered on the same date or data point in the other.

Update Chart Data

We sometimes have the need to split the data and then gracefully update the chart with the newly selected subset of data.

Modify Time Period

We sometimes have the need to view data for just the past n days. Here, the transition_on_update option is set to false.

Small Charts

At 150px, we change the font sizes for rollovers and axes.

Rollover Text Formatting

We can change the precision if the axis data type is a float. We can also change both the formatting, or hide the rollover text altogether.

Min / Max Numbers on Y Axis

Currently this lib will default to 0 on the y axis as min if there are no negative numbers. If there are negatives, should provide some buffer below.

Scatterplots

A work in progress, from styling to functionality.
