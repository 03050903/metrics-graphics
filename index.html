Metrics Graphics

Here is a collection, and something of a set of visual tests, of graphics we use for dashboards, reports, and other viz projects at Mozilla. The point of this repository is to ensure that everything under js/metrics-graphics.js works properly, so we may use that file for other graph-based projects. The main file, metrics-graphics.js, requires d3, jquery, and bootstrap at the moment. Below are a few examples. The data here is synthetic.
